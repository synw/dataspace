Initialize axis¶

Define the x and y axis¶

DataSpace.axis(x_axis_col: str, y_axis_col: str)

Set the columns to use for the chart x and y axis

Parameters

  • x_axis_col (str) – name of the column to use for x axis chart

  • y_axis_col (str) – name of the column to use for y axis chart

Example

ds.axis(“col1”, “col2”)

Define the chart engine¶

The default engine is Bokeh

DataSpace.altair() → None

Use the Altair charts engine

Example

ds.altair()

DataSpace.bokeh() → None

Use the Bokeh charts engine

Example

ds.bokeh()
