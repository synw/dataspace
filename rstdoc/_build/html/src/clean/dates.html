Dates¶

To date¶

DataSpace.to_date(*cols: str, **kwargs) → None

Convert some columns values to date type

Category: Clean/Dates/To date

Parameters

  • cols (str at least one) – names of the colums

  • **kwargs (optional) – keyword arguments for pd.to_datetime

Example

ds.to_date(“mycol”)

Format date¶

DataSpace.fdate(*cols, precision: str = 'S', format: Optional[str] = None)

Convert column values to formated date string

Parameters

  • *cols (str, at least one) – names of the colums

  • precision (str optional) – time precision: Y, M, D, H, Min S, defaults to “S”

  • format (str, optional) – python date format, defaults to None

Example

ds.fdate(“mycol1”, “mycol2”, precision=”D”)

Timestamps¶

DataSpace.timestamps(col: str, **kwargs)

Add a timestamps column from a date column

Parameters

  • col (str) – name of the timestamps column to add

  • **kwargs (optional) – keyword arguments for pd.to_datetime

Example

ds.timestamps("mycol")
