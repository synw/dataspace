Convert types¶

To integer¶

DataSpace.to_int(*cols: str, **kwargs) → None

Convert some column values to integers

Category: Clean/Convert types/To type

Parameters

  • *cols (str at least one) – names of the columns

  • **kwargs (optional) – keyword arguments for pd.to_numeric

Example

ds.to_int(“mycol1”, “mycol2”, errors=”coerce”)

To float¶

DataSpace.to_float(*cols: str, **kwargs) → None

Convert colums values to float -

Category: Clean/Convert types/To float

Parameters

  • cols (str at least one) – name of the columns

  • **kwargs (optional) – keyword arguments for df.astype

Example

ds.to_float(“mycol1”)

To type¶

DataSpace.to_type(dtype: type, *cols: str, **kwargs) → None

Convert colums values to a given type in the main dataframe

Category: Clean/Convert types/To type

Parameters

  • dtype (type) – a type to convert to: ex: str

  • *cols (str at least one*) – names of the columns

  • **kwargs (optional) – keyword arguments for df.astype

Examples

ds.to_type(str, "mycol")
