Nulls¶

Drop NaN¶

DataSpace.drop_nan(col: str, method: str = 'all', **kwargs) → None

Drop rows with NaN values from the main dataframe

Parameters

  • col (str optional) – name of the column

  • method (str optional) – how param for df.dropna, default: “all”

  • **kwargs (optional) – params for df.dropna

Example

ds.drop_nan(“mycol”)

Fill NaN¶

DataSpace.fill_nan(val: str, *cols)

Fill NaN values with new values in the main dataframe

Parameters

  • val (str) – new value

  • *cols (str at least one) – names of the colums

Example

ds.fill_nan("new value", "mycol1", "mycol2")

Fill nulls¶

DataSpace.fill_nulls(val=nan, *cols: str, nulls=[None, ''])

Fill all null values with NaN values in a column.

Null values are None or en empty string

Parameters

cols (str at least one) – columns names

Example

ds.fill_nulls(“mycol”)

Specify the values to be treated as nulls
