Values¶

Sort¶

DataSpace.sort(col: str, **kwargs)

Sorts the main dataframe according to the given column

Parameters

col (str) – column name

Example

ds.sort(“Col 1”)

Exclude¶

DataSpace.exclude(col: str, val) → None

Delete rows based on value

Parameters

  • col (str) – column name

  • val (any) – value to delete

Example

ds.exclude("Col 1", "value")

Dropr¶

DataSpace.dropr(*rows)

Drops some rows from the main dataframe

Parameters

rows (list of ints) – rows names

Example

ds.drop_rows([0, 2])

Append¶

DataSpace.append(*vals, ignore_index=True) → None

Append a row to the main dataframe

Parameters

  • vals (list) – list of the row values to add

  • index – index key, defaults to None

  • index – any, optional

Example

ds.append([0, 2, 2, 3, 4])

Reverse¶

DataSpace.reverse() → None

Reverses the main dataframe order

Example

ds.reverse()

Apply¶

DataSpace.apply(function, *cols: List[str], axis=1, **kwargs) → None

Apply a function on columns values

Parameters

  • function (function) – a function to apply to the columns

  • cols (name of columns) – columns names

  • axis – index (0) or column (1), default is 1

  • kwargs (optional) – arguments for df.apply

Example
def f(row):
        # add a new column with a value
        row["newcol"] = row["col1"] + 1
        return row

ds.apply(f)
